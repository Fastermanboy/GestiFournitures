☁️ Synchronisation Firebase

Ce fichier est désormais prêt pour Firebase Realtime Database. Une fois la configuration renseignée, toutes les données métiers (commandes, inventaires, comptes, messagerie, contrôle qualité, signatures et paramètres) sont synchronisées en temps réel via le cloud.

Mode local — Firebase non configuré
Aucune synchronisation distante active. Les données restent locales tant que Firebase n'est pas connecté.
Pré-requis Firebase : activez Realtime Database dans votre projet et, de préférence, Anonymous Authentication pour l'accès anonyme côté application.
Règles minimales conseillées pour un test :
{ "rules": { "workspaces": { "$workspace": { ".read": "auth != null", ".write": "auth != null" } } } }
Astuce : utilisez un espace de travail différent par environnement, par exemple savigny-prod, savigny-test ou savigny-recette.
